| Prop name | Description | Type |
| --- | --- | --- |
| align | Set the vertical aligment. [docs] | "bottom" | "middle" | "top" |

| caption | Put the <caption> on the top of the table. [docs] | "top" |
| className | A string of all className you want applied to the component. [docs] | string |
