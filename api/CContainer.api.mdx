| Prop name | Description | Type |
| --- | --- | --- |
| className | A string of all className you want applied to the base component. [docs] | string |
| fluid | Set container 100% wide, spanning the entire width of the viewport. [docs] | boolean |
| lg | Set container 100% wide until large breakpoint. [docs] | boolean |
| md | Set container 100% wide until medium breakpoint. [docs] | boolean |
| sm | Set container 100% wide until small breakpoint. [docs] | boolean |
| xl | Set container 100% wide until X-large breakpoint. [docs] | boolean |
| xxl | Set container 100% wide until XX-large breakpoint. [docs] | boolean |
