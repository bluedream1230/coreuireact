| Prop name | Description | Type |
| --- | --- | --- |
| className | A string of all className you want applied to the component. [docs] | string |
| narrow | Make sidebar narrow. [docs] | boolean |
| onHide | Method called before the hide animation has started. [docs] | () => void |
| onShow | Method called before the show animation has started. [docs] | () => void |
| overlaid | Set sidebar to narrow variant. [docs] | boolean |
| position | Place sidebar in non-static positions. [docs] | "fixed" | "sticky" |
| selfHiding | Make any sidebar self hiding across all viewports or pick a maximum breakpoint with which to have a self hiding up to. [docs] | boolean | Breakpoints |
| size | Size the component small, large, or extra large. [docs] | "sm" | "lg" | "xl" |
| unfoldable | Expand narrowed sidebar on hover. [docs] | boolean |
| visible | Toggle the visibility of sidebar component. [docs] | boolean |
