## Responsive images

Images in CoreUI for React.js are made responsive with `fluid` property. This applies `max-width: 100%;` and `height: auto;` to the image so that it scales with the parent element.

<Example>
  <CImage fluid src="/public/images/react.jpg" />
</Example>

```jsx
  <CImage fluid src="/public/images/react.jpg" />
```

## Image thumbnails

In addition to our [border-radius utilities](https://coreui.io/docs/4.0/utilities/borders), you can use prop`thumbnail` to give an image a rounded 1px border appearance.

<Example>
  <CImage rounded thumbnail src="/public/images/react400.jpg" width={200} height={200}/>
</Example>

```jsx
  <CImage rounded thumbnail src="/public/images/react400.jpg" width={200} height={200}/>
```

## Aligning images

Align images with the `align` property.

<Example>
  <div className="clearfix">
    <CImage align="start" rounded src="/public/images/react400.jpg" width={200} height={200}/>
    <CImage align="end" rounded src="/public/images/react400.jpg" width={200} height={200}/>
  </div>
</Example>

```jsx
  <div className="clearfix">
    <CImage align="start" rounded src="/public/images/react400.jpg" width={200} height={200}/>
    <CImage align="end" rounded src="/public/images/react400.jpg" width={200} height={200}/>
  </div>
```

<Example>
  <div className="clearfix">
    <CImage align="center" rounded src="/public/images/react400.jpg" width={200} height={200}/>
