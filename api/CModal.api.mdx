| Prop name | Description | Type |
| --- | --- | --- |
| alignment | Align the modal in the center or top of the screen. [docs] | "top" | "center" |
| backdrop | Apply a backdrop on body while modal is open. [docs] | boolean |
| className | A string of all className you want applied to the base component. [docs] | string |
| duration | @ignore | number |
| fullscreen | Set modal to covers the entire user viewport. [docs] | boolean | "sm" | "md" | "lg" | "xl" | "xxl" |
| keyboard | Closes the modal when escape key is pressed. [docs] | boolean |
| onDismiss | Method called before the dissmiss animation has started. [docs] | () => void |
| portal | Generates modal using createPortal. [docs] | boolean |
| scrollable | Create a scrollable modal that allows scrolling the modal body. [docs] | boolean |
| size | Size the component small, large, or extra large. [docs] | "sm" | "lg" | "xl" |
| transition | Remove animation to create modal that simply appear rather than fade in to view. [docs] | boolean |
| visible | Toggle the visibility of modal component. [docs] | boolean |
