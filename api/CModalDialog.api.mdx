| Prop name | Description | Type |
| --- | --- | --- |
| alignment | Align the modal in the center or top of the screen. [docs] | "top" | "center" |
| className | A string of all className you want applied to the base component. [docs] | string |
| fullscreen | Set modal to covers the entire user viewport. [docs] | boolean | "sm" | "md" | "lg" | "xl" | "xxl" |
| scrollable | Does the modal dialog itself scroll, or does the whole dialog scroll within the window. [docs] | boolean |
| size | Size the component small, large, or extra large. [docs] | "sm" | "lg" | "xl" |
